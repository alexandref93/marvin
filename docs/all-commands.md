# All commands

## append

Add more content to the end of the file.

### Params

| Param   | Description           | Type   |
|---------|-----------------------|:------:|
| file    | The file to append    | string |
| ...args | The content on append | string |

### Example

```
// .marvin

append file/text.txt hello world
```
